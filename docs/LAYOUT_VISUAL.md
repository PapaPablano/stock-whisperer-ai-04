## Layout Features

### Grid System
- **Desktop**: 3-column grid for main content
  - Chart: 2 columns (66.67%)
  - News: 1 column (33.33%)

- **Mobile**: Single column, stacked layout
  - Chart: 100% width
  - News: 100% width (below chart)

### Responsive Breakpoints
```css
/* Mobile First */
grid-cols-1              /* < 768px */

lg:col-span-1           /* News takes 1/3 */
```
